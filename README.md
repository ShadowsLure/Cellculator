## Introduction
As of 9th of August 2023, Cellculator project will be made partially public. We have featured [interactions.py](https://pypi.org/project/interactions.py/), a relatively new and a robust library for developing discord bots based on python.

The coding used is very simple and can be understood by most python enthusiast. We are making this public to recieve contribution which can potentially make the code even simpler and efficient

## What will we share?
I will only share the code that makes the bot work and not the logics involved. However, proper descriptions will be provided which should give an idea about the logics used.
I will also be not sharing our database updation files since those are relatively easy and a lot of community base is already out there to help. Our database is on [MongoDB](https://www.mongodb.com/atlas/database) which was made possible using the [pymongo](https://pypi.org/project/pymongo/) module.

This repo is meant to not only recieve contributions but to also help people who are starting out with [interactions.py](https://pypi.org/project/interactions.py/).

PS: Whenever a new logic is developed, i will share the older logic.
